# SmartHireX
SmartHireX is an AI-powered job portal built using Flask and SQLite.

## Features
- Job Seeker Registration & Login
- Employer Registration & Login
- Job Posting
- Job Application
- SQLite Database
- Flask Backend

## Technologies Used
- Python
- Flask
- SQLite
- HTML/CSS

## How to Run
1. Install requirements
2. Run: python app.py
3. Open http://127.0.0.1:5000
